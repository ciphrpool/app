# Standard Library

## Core Functions

### Error Handling
- `Err()` : Returns the error value of an Error type
- `Ok()` : Returns the success value of an Error type
- `assert(condition: bool)` : Verifies a condition, terminates the thread if false

### Memory Management
- `len(container: T) -> u64` : Returns the length of a container (String, Vec, Map)
- `cap(container: T) -> u64` : Returns the capacity of a container
- `free(ptr: &T)` : Deallocates memory allocated on the heap, Note that since vector, map, functions and string are already pointers there is no need to add '&' when freeing them
- `alloc(size: u64) -> &T` : Allocates memory on the heap
- `memcpy(dst: &T, src: &T, size: u64)` : Copies memory from source to destination
- `size_of(type: T) -> u64` : Returns the size in bytes of a type

### Vector Operations
- `vec(len : u64, cap: optional u64) -> Vec[Any]` : Creates a new empty vector, the capacity is optional
- `push(vec: Vec[T], item: T)` : Adds an item to the end of a vector
- `pop(vec: Vec[T]) -> T` : Removes and returns the last item of a vector
- `extend(vec: Vec[T], other: []T)` : Adds all items from other array
- `delete(vec: Vec[T], index: u64)` : Removes item at specified index
- `clear_vec(vec: Vec[T])` : Removes all items from vector

### Map Operations
- `map(capacity : optional u64) -> Map[K]V` : Creates a new empty map, the capacity is optional
- `insert(map: Map[K]V, key: K, value: V)` : Inserts key-value pair into map
- `delete_key(map: Map[K]V, key: K)` : Removes entry with specified key
- `get(map: Map[K]V, key: K) -> V` : Returns value associated with key
- `clear_map(map: Map[K]V)` : Removes all entries from map
- `free_map(map: Map[K]V)` : Deallocates map memory

### String Operations
- `string() -> String` : Creates a new empty string
- `append(str: String, other: String)` : Concatenates two strings
- `char_at(str: String, index: u64) -> char` : Returns character at specified index
- `to_str(value: T) -> String` : Converts value to string representation
- `clear_string(str: String)` : Removes all characters from string

### Format Conversions
- `format(value: str) -> String` : Formatted string, expressions are delimited by brackets
- `itoa(value: i64) -> String` : Converts integer to string
- `atoi(str: String) -> i64` : Converts string to integer
- `ftoa(value: f64) -> String` : Converts float to string
- `atof(str: String) -> f64` : Converts string to float
- `btoa(value: bool) -> String` : Converts boolean to string
- `atob(str: String) -> bool` : Converts string to boolean
- `ctoa(value: char) -> String` : Converts character to string
- `atoc(str: String) -> char` : Converts string to character

### Thread Management
- `spawn(func: fn()) -> u64` : Creates new thread running specified function
- `close(thread_id: u64)` : Terminates specified thread
- `exit()` : Terminates current thread
- `wait()` : Suspends current thread
- `wake(thread_id: u64)` : Resumes specified thread
- `sleep(seconds: u64)` : Suspends thread for specified duration
- `join(thread_id: u64)` : Waits for specified thread to finish

### Input/Output
- `print(value: String or str)` : Outputs value without newline
- `println(value: String or str)` : Outputs value with newline
- `printf(format: str)` : Formatted output, expressions are delimited by brackets
- `scan() -> String` : Reads input from standard input
- `to_str(value: T) -> String` : Converts value to string representation

### Iterators
- `items(map: Map[K]V>) -> Vec[(K,V)]` : Returns a vector with the keys and the values of the map
- `keys(map: Map[K]V) -> Vec[K]` : Returns a vector with the keys
- `values(map: Map[K]V) -> Vec[V]` : Returns a vector with the values

### Mathematical Functions
- `ceil(x: f64) -> f64` : Rounds up to nearest integer
- `floor(x: f64) -> f64` : Rounds down to nearest integer
- `abs(x: T where T is a numerical type) -> T` : Returns absolute value
- `exp(x: f64) -> f64` : Returns e raised to power x
- `log(x: f64, base: f64) -> f64` : Returns logarithm with specified base
- `ln(x: f64) -> f64` : Returns natural logarithm
- `log10(x: f64) -> f64` : Returns base-10 logarithm
- `pow(x: f64, y: f64) -> f64` : Returns x raised to power y
- `sqrt(x: f64) -> f64` : Returns square root

### Trigonometric Functions
- `acos(x: f64) -> f64` : Arc cosine
- `asin(x: f64) -> f64` : Arc sine
- `atan(x: f64) -> f64` : Arc tangent
- `atan2(y: f64, x: f64) -> f64` : Two-argument arc tangent
- `cos(x: f64) -> f64` : Cosine
- `sin(x: f64) -> f64` : Sine
- `tan(x: f64) -> f64` : Tangent
- `hypot(x: f64, y: f64) -> f64` : Hypotenuse
- `deg(rad: f64) -> f64` : Converts radians to degrees
- `rad(deg: f64) -> f64` : Converts degrees to radians

### Hyperbolic Functions
- `cosh(x: f64) -> f64` : Hyperbolic cosine
- `sinh(x: f64) -> f64` : Hyperbolic sine
- `tanh(x: f64) -> f64` : Hyperbolic tangent
- `acosh(x: f64) -> f64` : Inverse hyperbolic cosine
- `asinh(x: f64) -> f64` : Inverse hyperbolic sine
- `atanh(x: f64) -> f64` : Inverse hyperbolic tangent

### Constants and Special Values
- `is_nan(x: f64) -> bool` : Checks if value is Not a Number
- `is_inf(x: f64) -> bool` : Checks if value is infinite
- `pi() -> f64` : Returns Ï€ constant
- `e() -> f64` : Returns e constant
- `inf() -> f64` : Returns positive infinity
- `neg_inf() -> f64` : Returns negative infinity
